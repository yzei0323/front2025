css란

HTML이 웹 문서의 구조를 담당한다면
CSS 글꼴, 색생, 여백 같은 스타일을 추가하여
보기 좋게 꾸며주는 역할을 담당한다

CSS 적용방법

  • 인라인 스타일
  • 내부 스타일 , style태그
  • 외부스타일

선택자

  • 전체선택자 *
  • 태그선택자 tag
  • 클래스선택자 . : 중복허용
  • 아이디선택자 # : 한 개
  • 자손선택자 공백 (스페이스) : li .orange =>
  • 자식선택자 >
  • 인접형제선택사 +
  • 인접형제선택사 ~
  • 일치선택자 li.orange => li태그이면서 orange클래스를 가진 요소 선택
  • 가상클래스선택자 :hover :active :focus
  • 가상요소선택자 ::befor , ::after
