자기소개서

-소개-

OOO: 소프트웨어공학과를 전공하였고, 현재 프론트엔드 개발자를 목표로 공부 중입니다.

실력 향샹, 취업을 위해 에이콘아카데미 홍대학원에서 자바(JAVA) 웹개발자 양성과정(클라우드) 2회차를 수강 중입니다.

-사용 기술-

  • 프론트엔드
    • HTML
    • CSS
    • Javascript
    • React
  • 백엔드
    • JAVA
    • MVC2
    • Spring
  • 데이터베이스
    • Oracle
    • mySQL
  • 그 외 사용툴
    • Visual Studio Code
    • github
    • Docker
    • AWS 클라우드

-학력-

출신학교 전공(계열) 기간 학점(등급)
OO대학교 소프트웨어공학과 2019.03 ~ 2025.2 3.5/4.5
OO고등학교 이과 2016.03 ~ 2019.2 3.5
OO중학교 2013.03 ~ 2016.2
OO초등학교 2007.03 ~ 2013.2
