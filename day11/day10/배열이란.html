배열

많은량의 기억장소가 필요할 때
기억 단계와 처리단계 다를 때 배열 사용

간단한 문제로 배용사용 알아보기

60명학생의 성적을 입력받아 (이름, 국어,영어, 수학)서 학생평균을 출력하는 프로그램을 작성해라

추가사항 : 성적순으로 학생정보 출력 =>배열필요

추가사항 : 과목별 평균을 구하라

배열은 기준으로 부터 연속적으로 기억장소(변수)를 확보 하는 것

배열은 기억장소를 배열의 요소라고 한다

배열의 요소에 접근하기 위해서 인덱스(첨자)를 사용한다.

인덱스는 0부터 시작한다.

인덱스로 변수를 사용할 수 있다 ( 정수형 값)
